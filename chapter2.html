Chapter 2: Elements

1. Structural Elements

These elements define the basic structure of a webpage.

<html>
<head>
<body>
</body>
</html>

2. Text Elements

Used to display and format text on the webpage.

<h1>Heading 1</h1>
<p>This is a <strong>strong</strong> paragraph.</p>

3. Link Element

Used for creating hyperlinks.

<a href="https://www.example.com">Visit Example</a>

4. Image Element

Used for embedding images.

<img src="image.jpg" alt="Image Description">

5. Form Elements

Used to create forms for user input.

<form>
<input type="text" name="name">
<button>Submit</button>
</form>

6. Table Elements

Used to create tables.

<table>
<tr><th>Header</th></tr>
</table>

7. List Elements

Used to create ordered and unordered lists.

<ul><li>Item 1</li></ul>
<ol><li>First</li></ol>

8. Semantic Elements

Elements that provide meaning to the content.

<header>Header Content</header>
<footer>Footer Content</footer>

9. Media Elements

Used to embed audio and video content.

<audio controls><source src="audio.mp3" type="audio/mp3"></audio>
<video controls><source src="video.mp4" type="video/mp4"></video>

10. Script and Style Elements

For embedding JavaScript and CSS styles.

<style> body { background-color: #f0f0f0; } </style>
<script> alert("Hello!"); </script>
