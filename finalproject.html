Final Project: Build a Domain-Based Website

Choose a domain you're interested in and create your own web page project!

Available Domains

  • Education - E.g., Online Course Page
  • Health - E.g., Fitness Tracker Form
  • Travel - E.g., Trip Booking Form
  • E-Commerce - E.g., Product Order Page
  • Food - E.g., Online Food Order
  • Entertainment - E.g., Movie Ticket Booking
  • Portfolio - E.g., Personal Resume Website
  • Blog - E.g., Travel Blog Page

Project Instructions

  • Choose any domain of your interest from above
  • Create a web page using HTML
  • Include headings, paragraphs, lists, forms, images, links, and interactivity
  • Make sure the content is clear and well-structured
  • Write a short description about your project idea
