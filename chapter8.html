Chapter 8: Creating Tables in HTML

Tables in HTML are used to display data in a structured format using rows and columnsâ€”like a spreadsheet or data grid. This is helpful for showing schedules, price lists, results, or any information that fits a tabular form.

Why Use Tables?

  • Compare different pieces of data side-by-side.
  • Present structured data in a readable format.
  • Create complex layouts (though CSS is preferred for layout today).

Basic Table Structure

<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Row 1, Col 1</td>
    <td>Row 1, Col 2</td>
  </tr>
</table>

Explanation of Tags

TagMeaningUse
<table>Table elementWraps the entire table
<tr>Table RowDefines a row in the table
<th>Table HeaderDefines a heading cell
<td>Table DataDefines a normal data cell

Notes:
  • Use <thead>, <tbody>, and <tfoot> for better structure in large datasets.
  • Use CSS Grid or Flexbox for page layout, not tables.
  • Make tables responsive using JavaScript or media queries.
