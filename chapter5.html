Chapter 5: Adding Links in HTML

What are Links?

Links (or anchor tags) in HTML allow users to click and navigate to:

  • External websites
  • Other pages of your website
  • Download files
  • Open email programs
  • Jump to a specific part of the same page

Bonus Tip: Styling Links

You can style links using CSS like this:


a {
  color: blue;
  text-decoration: none;
}
a:hover {
  text-decoration: underline;
}
